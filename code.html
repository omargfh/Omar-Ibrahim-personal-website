printf("Hello, world!");

With computer coding, I feel like Houdini. My stage is my laptop monitor as I active Browser Sync or Visual Studio Code. Whether it is a printf("Hello, world!" ); from my first computer science course or a complicated 400-line code to create an advance spell-checker at minimal runtime in C, every line of code has a tangible effect. Although many find coding boring, I find it as fascinating as every refresh or run command I use to see my code growing before my eyes.

I started programming in early 2018 (or what I call programming at least). Before that, I have been an amused wanderer looking behind glass or just a tip-toe dancer sinking the tips of my toes in before rushing them out quickly (and I did that a lot). My first programming language was C++ from an online blog, followed by basics of C online (I know, weird timeline). However, my first actual experience, that of 2018, was with Javascript on Code Academy and Khan Academy, followed by Swift through CodeWithChris YouTube channel. Watching Swift content exposed me to the ideas of model-view-control patters, libraries, frameworks, UI design, handling events, and creativity of design.

Now, I work with HTML, CSS, Javascript, Python, C, C++, MySql, PHP, and more. So, welcome friend to the stage of The Great Houdini.
